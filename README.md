# RSV Trailers — Website

Custom food trailer manufacturer based in Monterrey, Mexico serving the US market.

**Live site:** [rsvtrailers.com](https://rsvtrailers.com)

## Stack

- Single-page HTML/CSS/JS — no frameworks, no build tools
- Hosted on Netlify (auto-deploys from this repo)
- Images served as WebP with JPG fallbacks

## Structure

```
/
├── index.html        ← full website (HTML + CSS + JS in one file)
└── images/
    ├── hero.webp
    ├── galeria-1.webp / .jpg
    ├── galeria-2.webp / .jpg
    ├── galeria-3.webp / .jpg
    └── galeria-4.webp / .jpg
```

## Making changes

Edit `index.html`, commit, and push to `main` — Netlify deploys automatically.
